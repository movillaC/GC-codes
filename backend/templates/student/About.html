About This Project

This app is a personal project I built to practice my skills as an IT student. My goal was to take what I've learned in the classroom and apply it to a real, working application.

Why I Built This

From the initial logic and database structure to the final visual look, it is a way for me to bridge the gap between technical coding and clean design.

My Goal

Since I am still building my skills, this project is all about learning. It represents my progress in:

Development: Writing clean, functional code.
Design: Creating a simple and clear user interface.
Problem Solving: Figuring out how to turn an idea into a digital tool.

Thank you for checking out my work!
